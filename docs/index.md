# Nativetalk Call SDK

Plug-and-play SIP/VoIP calling SDK for React Native, powered by Linphone.

## Features

- 📞 **SIP Registration & Calls** — Full VoIP support via Linphone 5.x
- 🎛️ **Call Control** — Answer, dial, hold, mute, speaker control, DTMF
- 📱 **UI Components** — Pre-built Dialer, IncomingCallView, OutgoingCallView
- 🔧 **Easy Setup** — Expo config plugin handles native setup automatically
- 📋 **Call Logs** — Track call history with metadata
- 🌐 **Multi-tenant** — API key + JWT authentication for secure integration

## Quick Start

```bash
npm install @nativetalkcommunications/react-native-call-sdk
```

Then follow the [Quickstart Guide](quickstart.md).

## Integration

For Facility Bills CRM integration:

1. **Install SDK** — `npm install @nativetalkcommunications/react-native-call-sdk`
2. **Configure** — Set your SIP domain and credentials via the [Configuration](configuration.md) guide
3. **Manage Devices** — Use the [SIP Device API](SIP_DEVICE_API.md) to programmatically create and manage SIP devices
4. **Implement Dialer** — Follow the [Quickstart](quickstart.md) to integrate calling UI
